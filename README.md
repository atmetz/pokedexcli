# pokedexcli

REPL and cacheing system for PokeAPI(https://pokeapi.co/docs/v2) written in Go.

Usage:

pokedex: See a list of all caught pokemon
exit: Exit the Pokedex
help: Displays a help message
map: Get the next page of locations
mapb: Get the previous page of locations
explore <location_name>: Explore a location
catch <Pokemon name>: Attempt to chatch a pokemon
inspect <Pokemon name>: See the stats of a caught pokemon
